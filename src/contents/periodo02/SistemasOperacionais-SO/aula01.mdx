# Aula 01: Introdução aos Sistemas Operacionais

Bem-vindo à primeira aula de **ADS**. Hoje vamos entender o que é o Kernel e como ele gerencia o hardware.

## O que é um Sistema Operacional?

O SO é a camada entre o hardware e o usuário. Sem ele, você teria que escrever código binário para cada clique no mouse.

### Exercício de Fixação

Qual é a principal função do Kernel em um sistema operacional?

A) Gerenciar apenas a interface gráfica.\
B) Servir como o núcleo do sistema, gerenciando CPU, memória e dispositivos.\
C) Navegar na internet.
