# Configurando Nginx

## Baixando Dependências

Vá para o site do Ubuntu e baixe a versão [server install image](https://releases.ubuntu.com/resolute/).

## Configurando VirtualBox

Dentro do aplicativo VitualBox, clique em **Novo** para a importação da imagem.

### Página "Virtual machine name and operating system"

Dê um nome para a VM no campo `VM Name` e importe a imagem baixada no campo `ISO Image` e em **Outro...**. Em seguida, clique em `Próximo (N)`.

### Página "Set up unattended guest OS installation"

Dentro do campo `User Name and Password`, escolha um nome e uma senha para a Máquina Virtual. Exemplo:

- User Name: ads
- Password: 654123

Depois de escolher o Nome e a Senha, pode apertar em `Próximo (N)`.

### Página "Specify virtual hardware"

Nessa página, você irá configurar as especificidades do hardware da Máquina Virtual.

- Base Memory: 4096MB
- Numbers of CPUs: 2
- Disk Size: 25,00 GB

Depois de configurar, pode apertar em `Próximo (N)`.

### Página "Sumário"

Nessa página, será informado um sumário geral das informações da Máquina Virtual para serem conferidas. Pode apertar em `Próximo (N)`.

O VirtualBox irá abrir automaticamente a VM configurada, abrindo uma aba com o título com a estrutura de "_UserName \[Executando\] - Oracle VirtualBox_" e instalando automaticamente o Ubuntu.

OBS: cuidado para não dar **Alt + Tab** dentro da VM durante a instalação.

## Dentro da VM

Quando a instalação estiver concluída, aparecerá uma tela com a versão do Ubuntu que foi baixada e irá pedir o login informado na configuração.

- Nome: ads
- Senha: 654123

# Mover Certificados Digitais para uma pasta

Esses certificados podem ser usados para a atividade 01, sem a necessidade de recriá-los. Para fazer isso, siga o passo a passo abaixo:

<Terminal>
    <Abas>
        <Aba active>Docker</Aba>
    </Abas>
    <Comentario>// criando a pasta</Comentario>
    <Prompt>{"root@24c166e57434:/var/www/html#"}</Prompt> mkdir atv1
    <Comentario>// movendo todos os arquivos relacionados ao usuário para a pasta</Comentario>
    <Prompt>{"root@24c166e57434:/var/www/html#"}</Prompt> mv djr.\* atv1
    <Comentario>// verificando os arquivos existentes</Comentario>
    <Prompt>{"root@24c166e57434:/var/www/html#"}</Prompt> ls
    <Comentario>// resultado esperado</Comentario>
    <Resultado>atv1 daladierjr.key daladierjr.pem index.html index.nginx-debian.html</Resultado>
    
</Terminal>

- (root@24c166e57434:/var/www/html#) mkdir atv1 // criando a pasta
- (root@24c166e57434:/var/www/html#) mv djr.\* atv1 // movendo todos os arquivos relacionados ao usuário para a pasta
- (root@24c166e57434:/var/www/html#) ls
// resultado esperado
atv1, daladierjr.key daladierjr.pem index.html index.nginx-debian.html

(root@24c166e57434:/var/www/html#) openssl req -x509 -nodes -days 365 -newkey rsa:2048 -extensions v3_ca  -config /etc/ssl/openssl.cnf -keyout djr.kay -out djr.pem

// escolha as opções para a criação do nome do domínio

// nos arquivos a seguir, tire os parâmetros das linhas que iniciam com ""
